# 여비온 — 관외출장 여비 정산 시범 프로그램

추가 유료 서비스 없이 실행할 수 있는 로컬 웹 프로그램입니다. 출장 정보, 여비 자동 산정, 차량 종류별 기준연비, 오피넷 유가 조회, 실비 입력, 최종금액 원단위 절사, 증빙 첨부, 총괄 정산서 PDF 출력 기능을 포함합니다.

## 실행

가장 간단한 방법은 프로젝트 폴더의 `여비온 실행.cmd`를 더블클릭하는 것입니다. 백그라운드 서버가 준비된 것을 확인한 후 브라우저가 열립니다.

직접 실행하려면 다음 명령을 사용합니다.

```powershell
npm start
```

브라우저에서 `http://127.0.0.1:4173`을 엽니다.

`public/index.html`을 직접 열면 화면 스타일은 표시되지만 저장·첨부·오피넷 기능은 작동하지 않습니다. 반드시 위 실행 파일 또는 서버 실행 명령을 사용해 주세요.

## GitHub Pages

`main` 브랜치에 푸시하면 GitHub Pages 배포 워크플로가 `public/` 폴더를 배포합니다. Pages 버전은 각 브라우저의 로컬 저장소에 출장 정보와 증빙자료를 보관하므로, 다른 기기와 데이터가 공유되지 않습니다. 첨부파일은 건별 2MB까지 가능하며 오피넷 자동 조회 대신 관리자 기준단가를 사용합니다.

## 운영 전 확인사항

- 관리자 설정에서 실제 여비 규정·단가·숙박비 상한을 회계부서 기준으로 확정합니다.
- 오피넷 무료 API 인증키를 발급받아 관리자 설정에 등록합니다.
- 현재 버전은 로컬 시범운영용이며, 실운영 시 로그인·권한분리·개인정보 암호화·백업·전자결재 연계를 추가해야 합니다.
- 첨부 이미지에는 OCR을 적용하지 않으며, PDF 출력 시 증빙자료 사진대지에 자동 배치됩니다. PDF 첨부파일은 원본 파일로 별도 보관됩니다.
- 철도·버스·숙박비는 실제 결제액을 입력하고 관련 증빙자료를 첨부합니다.
- 최종 지급액은 전체 인정금액을 합산한 뒤 10원 미만 원단위를 절사합니다.

## 데이터 보관

- 업무 데이터: `data/store.json`
- 첨부파일: `data/uploads/`

이 폴더는 실행 시 자동 생성됩니다.
